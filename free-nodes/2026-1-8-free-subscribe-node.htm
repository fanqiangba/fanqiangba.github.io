三、手把手安装配置教程

3.1 准备工作

首先确保你的Chrome浏览器是最新版本（目前最新为v115+）。点击浏览器右上角的三个点>帮助>关于Google Chrome即可检查更新。我建议在安装前先清理浏览器缓存，这能避免一些潜在的兼容性问题。

3.2 详细安装步骤

  1. 打开Chrome网上应用店（可直接搜索"Chrome Web Store"）
  2. 在搜索框输入"蔷薇科学上网"或"Rose VPN"
  3. 找到官方扩展（注意识别开发者信息，避免安装山寨版本）
  4. 点击"添加到Chrome"按钮
  5. 在弹出的权限请求对话框中点击"添加扩展程序"

专业提示：如果无法访问Chrome应用商店，可以尝试从开发者官网下载CRX文件，然后通过"开发者模式"手动安装。

3.3 首次配置指南

安装完成后，浏览器右上角会出现蔷薇图标。点击后你会看到一个简洁的控制面板：

  • 服务器选择：建议优先选择地理位置较近的节点（如日本、新加坡对亚洲用户较友好）
  • 协议设置：高级用户可切换WireGuard或OpenVPN协议
  • 自动连接：可设置特定网站自动启用代理

我个人的配置经验是：白天使用日本节点获得低延迟，晚上切换至美国节点获取更稳定带宽。对于视频流媒体，建议选择专用流媒体优化节点。

四、高级使用技巧与性能优化

4.1 分流策略配置

通过右键点击扩展图标>选项，可以设置分流规则： - 全局代理：所有流量通过VPN - 智能分流：仅国外网站走代理 - 自定义规则：指定域名列表

我通常选择智能分流模式，这样国内网站依然保持直连速度，而国际流量得到保护。

4.2 多账号管理

支持同时添加多个订阅账号，通过点击用户头像切换。这对需要区分工作和个人使用的用户特别有用。

4.3 速度优化技巧

  • 在拥挤时段尝试较少人使用的服务器
  • 更换连接协议（WireGuard通常最快）
  • 禁用其他占用带宽的扩展
  • 清理浏览器缓存和cookies

FAQ

Mellow 如何通过配置实现多出口分流？
Mellow 可在配置文件中定义多个 outbound，并结合 routing 规则实现按域名、IP 或策略组分流。这样不同类型的流量可走不同节点，提高使用体验。
Mihomo 节点分组功能有什么作用？
节点可按地区、延迟或用途分类，结合策略组自动分流流量。便于管理大量节点，提高访问速度和连接稳定性，同时优化维护效率，方便用户快速选择最优节点。
Mihomo 节点分组如何提升访问效率？
节点可按延迟、地区或用途分组，结合策略组实现自动分流。快速选择最优节点，可提高网页、视频和游戏访问速度，同时优化节点管理和维护效率，提升整体网络体验。
V2Ray QUIC 传输适合什么网络环境？
QUIC 基于 UDP 支持多路复用和零握手，降低延迟，提高吞吐量。适合高丢包、不稳定网络环境，可保证视频、游戏和即时通讯流量的稳定性和流畅体验。
Shadowrocket 支持哪几种代理协议？
Shadowrocket 支持多种协议，包括 Shadowsocks、Vmess、VLess、Trojan、HTTP、SOCKS5 等。它还可解析部分 QuantumultX 与 Clash 的订阅格式，适合 iOS 用户进行全协议代理管理与切换。
Mellow 如何配置透明代理？
Mellow 通过 tun2socks 技术实现透明代理。只需在配置文件中启用该功能，系统所有网络流量都会自动转发到代理内核，用户无需手动设置系统代理参数。
VMess TLS 与 WebSocket 组合有哪些优势？
TLS + WebSocket 可将 VMess 流量伪装为 HTTPS，隐藏真实通信特征。结合策略组使用，可增强抗封锁能力，提高节点稳定性和访问速度，特别适合视频和游戏场景。
如何在路由器上使用 PassWall2 为不同设备设置不同策略？
在 PassWall2 的规则或策略组中为固定 IP 或 MAC 地址创建特定路由条目，将其映射到不同的策略组（如工作机走直连、游戏机走低延迟节点），并保存规则后重载以生效。
Mellow 的 TUN 模式与系统全局代理有何区别？
TUN 模式直接拦截 IP 层流量，不依赖应用代理设置，系统全局代理通常只影响支持 HTTP/HTTPS 的程序。TUN 更适合全局科学上网和游戏环境。
WinXray 如何实现延迟最低节点自动切换？
WinXray 支持节点延迟检测和策略组配置，可自动将流量切换至延迟最低的节点。用户无需手动选择，提高游戏、视频和网页访问速度及稳定性，优化整体网络体验。
